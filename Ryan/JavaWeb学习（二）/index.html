四、JavaScript

一 JS简介

1.1什么是 JavaScript？

JavaScript（简称 JS）是一种轻量级、解释型、动态的编程语言，主要用于网页开发，使网页具备交互性，如动态更新内容、表单验证、动画效果等。

1.2 JavaScript 的主要特点

  1. 脚本语言：无需编译，浏览器可以直接运行。
  2. 跨平台：可以在所有主流浏览器（Chrome、Firefox、Edge 等）上运行。
  3. 弱类型：变量可以存储不同类型的数据，不需要声明类型。
  4. 基于对象：提供了对象（Object）、数组（Array）等数据结构。
  5. 事件驱动：可以响应用户操作，如点击、输入等。
  6. 动态网页：结合 HTML 和 CSS，可以创建交互式网页。

二 JS的数据类型和运算符

2.1 JS的数据类型

JavaScript 的数据类型可以分为 两大类：

  1. 基本类型（值类型 / 原始类型）
  2. 引用类型（对象类型）

1. 基本数据类型（Primitive Types）

基本数据类型存储的是值，它们不可变，并且存储在栈（Stack）内存中。包括：

类型 描述 示例
Number 数字类型（整数 & 小数） let a = 10; let b = 3.14;
String 字符串类型（文本） let name = "Ryan";
Boolean 布尔值（true / false） let isOk = true;
Undefined 变量已声明但未赋值 let x;（x 的值是 undefined）
Null 空值，表示“无” let y = null;
Symbol 唯一值（ES6 引入） let s = Symbol("唯一值");
BigInt 任意长度的整数（ES11 引入） let big = 123n;

示例

2. 引用数据类型（Reference Types）

引用类型存储的是对象的地址（指针），存储在堆（Heap）**内存中。包括：

类型 描述 示例
Object 对象 let obj = { name: "Ryan", age: 20 };
Array 数组 let arr = [1, 2, 3];
Function 函数 function sayHello() { console.log("Hello"); }
Date 日期对象 let now = new Date();
